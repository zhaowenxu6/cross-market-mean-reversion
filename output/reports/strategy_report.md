## 1. 方法论

### 1.1 因子模型
- **方法**: Ridge回归（α=0.01），滚动窗口90天，步长22天
- **因子**: BTC, ETH, SPY, QQQ, SMH（5个系统性因子）
- **残差提取**: r_target = α + β₁·r_factor₁ + β₂·r_factor₂ + ε
- **模型质量**: R²中位数0.535，15/25资产R²≥0.5
- **加分项**: 使用Ridge处理因子共线性（vs OLS）

### 1.2 信号生成
- **入场**: |z| > 2.0（残差超过2σ）
- **平仓**: |z| < 0.5（残差回归正常范围）
- **止损**: |z| > 4.0（回归失败）
- **z-score窗口**: 30天滚动μ, σ

### 1.3 组合构建
- **风险预算**: N = (σ_target / σ_pair) × TOTAL_CAPITAL, σ_target=5%
- **单资产上限**: 3%
- **板块上限**: 15%
- **杠杆上限**: 3x
- **因子净暴露上限**: 单因子5%

### 1.4 成本建模
- 手续费: 现货0.10%, Perp 0.04%（taker）
- 滑点: 主流0.03%, 山寨0.05%, 美股0.02%, 币股0.15%
- 资金费率: Binance 真实历史数据
- 借券成本: 年化2%（做空美股）
- 保证金机会成本: 年化4%
- 极端事件: SVB/ETF/SEC/日元套利期间滑点×3

## 5. 策略局限与改进方向

### 5.1 当前策略的主要风险点

1. **Alpha不足以覆盖成本**: 日频残差回归信号的预测力太弱，每笔交易产生的毛收益（均值~$0-$100）无法覆盖$~180的交易成本
2. **残差回归速度与采样频率不匹配**: 残差半衰期分析显示所有资产半衰期<0.5天，意味着回归发生在日内而非日间。日频回测无法捕捉此类快速回归
3. **低信号频次**: ±2σ阈值仅产生日均0.8个新信号，组合分散化程度不足（日均持仓1.8-2.1个）
4. **方向一致性偏误**: 加密资产多配置BTC+ETH因子，同向信号时因子净暴露快速触顶

### 5.2 改进方向

1. **高频数据（首要改进）**: 使用4小时K线替代日线，匹配残差的快速回归特性
2. **精选标的**: 回测显示ARB、MSFT、AMZN、NVDA等资产具有正期望收益，可缩减标的池
3. **动态因子选择**: 用LASSO自动选择各资产的最优因子组合，而非固定因子配置
4. **交易成本优化**: 部分订单以maker挂单成交（费率降低50%+）
5. **市场状态过滤**: 在高波动率环境中放宽入场阈值，在低波动率中收紧

### 5.3 长期可持续性评估

该策略的alpha来源是跨市场定价偏差——加密用户对美股/币股的情绪化定价。随着Binance等平台跨市场交易功能的成熟，这类定价偏差预计会逐渐缩小。3年后的可行性取决于：

- **Alpha衰减**: 预计每年衰减10-20%，更多套利者进入会压缩利润空间
- **成本下降**: 交易所费率竞争可能降低交易成本，对冲alpha衰减
- **适应路径**: 更高的数据频率和更精细的风险管理是持续保持竞争力的关键
